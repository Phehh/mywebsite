# Nichol Pheh Trading Website

## 📖 Overview
Nichol Pheh Trading is a web design company dedicated to helping businesses establish a strong online presence.  
This repository contains the source code for the official business website, built with HTML, CSS, and JavaScript.

## 🏗️ Project Structure
- `index.html` → Homepage with business description, services, payment info, contact details, and promotional content.
- `style.css` → Styling for the website.
- `script.js` → JavaScript for interactivity.
- `terms.html` → Legal page with Terms & Conditions.

## 🌐 Features
- Responsive design for desktop and mobile.
- Clear sections for About, Services, Payments, Contact, and Promotions.
- Dedicated Terms & Conditions page.
- Social media links for easy engagement.

## 🚀 How to Run
1. Clone or download this repository.
2. Open the folder in **Visual Studio Code**.
3. Install the **Live Server** extension.
4. Right-click `index.html` → **Open with Live Server**.
5. The website will open in your browser.
